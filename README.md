# PakiPark Login

## Open the Login Page

1. Open PowerShell and go to the frontend folder (depending on where you saved it):  
   `cd D:\PakiPark-Login-SignUp\PakiPark-Login-SignUp\LoginSignUpCode\frontend` OR  
   `cd C:\PakiPark-Login-SignUp\PakiPark-Login-SignUp\LoginSignUpCode\frontend`
2. Install dependencies: `npm install`
3. Start the dev server: `npm run dev -- --port 3000`
4. Open in browser: `http://localhost:3000/login` (you can also use `http://localhost:3000/`)
